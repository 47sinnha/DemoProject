# 🛒 Enterprise Retail Sales Analytics

An end-to-end Business Intelligence portfolio project demonstrating SQL-based data analysis and interactive Power BI dashboard development for retail sales performance.

---

## 📌 Project Overview

This project analyzes retail sales data to uncover valuable business insights related to sales, profit, customer behavior, discount impact, and regional performance.

The complete workflow includes:

- Data Cleaning & Validation
- SQL Analysis
- SQL View Creation
- KPI Development
- Interactive Power BI Dashboard
- Business Insights & Recommendations
