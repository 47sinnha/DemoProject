# SQL Analysis

This folder contains SQL scripts used to analyze the retail sales dataset and prepare optimized views for Power BI reporting.

## Contents

- KPI Queries
- Customer Analysis
- Category Analysis
- Regional Analysis
- Monthly Revenue Trends
- Discount Analysis
- SQL Views for Dashboard

## SQL Concepts Used

- SELECT
- GROUP BY
- ORDER BY
- Aggregate Functions
- CASE Statements
- Window Functions
- Views
- Ranking Functions

The SQL scripts are designed to transform raw transactional data into meaningful business insights and support interactive dashboard reporting.
